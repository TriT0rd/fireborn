# FireBorn Theme

A custom Shopify theme for FireBorn Fashion Brand.

## Color Palette
- Midnight Navy: #0A1128
- Blaze Orange: #FF4500
- Steel Grey: #8C92AC
- Crisp White: #FFFFFF

## Fonts
- Bebas Neue (Display)
- Barlow Condensed (UI)
- Inter (Body)

## Setup
See installation checklist in theme documentation.

## Development
Built with native Shopify Liquid + CSS + Vanilla JS.
No frameworks. No build tools required.
